# 📊 셀록홈즈 자동검색 확장 설치 가이드

구매대행 소싱 대시보드에서 **📊 셀록홈즈** 버튼을 누르면,
셀록홈즈가 열리면서 **키워드가 자동으로 검색**되게 해주는 확장입니다.

> 이 확장을 설치하지 않아도 앱은 정상 작동합니다.
> (미설치 시에는 키워드가 자동 복사되어, 검색창에 Ctrl+V 후 Enter로 검색하면 됩니다.)

---

## 설치 방법 (최초 1회, 약 2분)

### 1단계 — Tampermonkey 설치
크롬 웹스토어에서 **Tampermonkey**(무료)를 설치합니다.

1. 크롬에서 아래 주소 접속
   `https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo`
2. **"Chrome에 추가"** 클릭 → **"확장 프로그램 추가"** 클릭

### 2단계 — 자동검색 스크립트 등록
1. 크롬 우측 상단 **퍼즐 조각 아이콘(확장)** → **Tampermonkey** 클릭
2. **"새 스크립트 만들기"** 클릭
3. 편집창의 내용을 **모두 지우고**, 아래 파일의 내용을 **전체 복사해서 붙여넣기**
   - 파일: `셀록홈즈_자동검색.user.js`
4. 상단 메뉴 **파일 → 저장** (또는 Ctrl+S)

### 3단계 — 확인
1. 소싱 대시보드에서 **📊 셀록홈즈** 버튼 클릭
2. 셀록홈즈가 새 탭에서 열리면서 **해당 키워드가 자동으로 검색**되면 성공! 🎉

---

## 자주 묻는 질문

**Q. 자동검색이 안 돼요.**
- Tampermonkey가 **켜져 있는지**(활성화) 확인하세요.
- 셀록홈즈에 **로그인**되어 있어야 합니다.
- 스크립트를 붙여넣을 때 **맨 윗줄(`// ==UserScript==`)부터 끝까지** 전부 복사했는지 확인하세요.

**Q. 안전한가요?**
- 이 스크립트는 셀록홈즈 키워드분석 페이지에서만 작동하며,
  URL의 키워드를 검색창에 입력하고 Enter를 누르는 것 외에는 아무 동작도 하지 않습니다.

**Q. 검색 사용량이 차감되나요?**
- 자동검색도 일반 검색과 동일하게 셀록홈즈의 월 사용량을 1회 차감합니다.
  (직접 검색하는 것과 동일)
