# 📊 셀록홈즈 자동검색 확장 설치 방법

구매대행 소싱 대시보드에서 **📊 셀록홈즈** 버튼을 누르면,
셀록홈즈가 열리면서 **키워드가 자동으로 검색**되게 해주는 크롬 확장입니다.
(Tampermonkey 같은 별도 프로그램 필요 없음)

> 이 확장을 설치하지 않아도 앱은 정상 작동합니다.
> (미설치 시에는 키워드가 자동 복사되어, 검색창에 Ctrl+V 후 Enter로 검색)

---

## 설치 방법 (최초 1회, 약 1분)

### 1단계 — 확장 폴더 준비
`셀록홈즈_자동검색_확장` 폴더를 원하는 위치에 둡니다.
(이 폴더 안에 `manifest.json`, `content.js` 가 있어야 합니다. **폴더째로** 두세요.)

### 2단계 — 개발자 모드 켜기
1. 크롬 주소창에 입력 후 Enter:  `chrome://extensions`
2. 오른쪽 위 **"개발자 모드"** 스위치를 **켜기(ON)**

### 3단계 — 확장 불러오기
1. 왼쪽 위 **"압축해제된 확장 프로그램을 로드합니다"** 버튼 클릭
2. `셀록홈즈_자동검색_확장` **폴더를 선택** (폴더 안 파일이 아니라 폴더 자체)
3. 목록에 **"셀록홈즈 URL 자동검색"** 이 추가되면 완료! 🎉

### 4단계 — 확인
1. 소싱 대시보드에서 **📊 셀록홈즈** 버튼 클릭
2. 셀록홈즈가 새 탭에서 열리면서 **해당 키워드가 자동 검색**되면 성공!

---

## 자주 묻는 질문

**Q. "개발자 모드 확장 프로그램을 사용 중지" 알림이 떠요.**
- 크롬을 켤 때 가끔 뜨는 정상 알림입니다. **"확장 프로그램 유지"** 를 누르면 계속 사용됩니다.

**Q. 자동검색이 안 돼요.**
- `chrome://extensions` 에서 이 확장이 **켜져 있는지** 확인하세요.
- 셀록홈즈에 **로그인**되어 있어야 합니다.
- 폴더를 옮겼다면 `chrome://extensions` 에서 새로고침(↻) 하거나 다시 로드하세요.

**Q. 폴더를 삭제하면 어떻게 되나요?**
- 확장이 작동을 멈춥니다. 폴더는 지우지 말고 그대로 두세요.

**Q. 검색 사용량이 차감되나요?**
- 자동검색도 일반 검색과 동일하게 셀록홈즈 월 사용량을 1회 차감합니다. (직접 검색과 동일)
