🚀 구매대행 소싱 도구 · 시작 가이드

① 쿠팡 소싱 키워드 필터링  ·  ② 중국어 상세페이지 이미지 번역

1단계 · Python 설치 (최초 1회)

1
https://python.org 접속 → Download Python 클릭
2
설치 화면에서 "Add Python to PATH" 체크 후 Install
(Microsoft Store에서 "Python 3" 검색 후 설치해도 됩니다)
💡 이미 설치돼 있는지 모르겠다면 그냥 2단계로 넘어가세요. install.bat이 자동으로 확인하고 없으면 안내해줍니다.

2단계 · 라이브러리 설치 (최초 1회)

  1. 받은 ZIP 파일의 압축을 풉니다
  2. 폴더 안 install.bat 을 더블클릭
  3. 검은 창에서 설치가 진행됩니다 (2~3분)
  4. "설치가 완료되었습니다!" 메시지가 뜨면 아무 키나 눌러 닫기

3단계 · 앱 실행 (매번)

  1. run.bat 을 더블클릭
  2. 잠시 후 브라우저가 자동으로 열립니다 (안 열리면 주소창에 localhost:8501 입력)
  3. 사용이 끝나면 검은 창을 닫아 종료
⚠️ 검은 창(터미널)은 닫지 마세요 — 닫으면 앱이 종료됩니다.

왼쪽 사이드바에 두 페이지가 있습니다: 🔍 구매대행 소싱 대시보드 / 🌐 중국어 → 한국어 이미지 번역

4단계 · 크롬 확장프로그램 설치 (최초 1회)

복사·붙여넣기 없이 키워드를 셀록홈즈 안에서 자동 검색해주는 프로그램입니다.

  1. 셀록홈즈_자동검색_확장 폴더를 원하는 위치에 둡니다 (폴더째로).
  2. 크롬 주소창에 chrome://extensions 입력
  3. 오른쪽 위 개발자 모드 켜기
  4. "압축해제된 확장 프로그램을 로드합니다" → 셀록홈즈_자동검색_확장 폴더 선택 (폴더 안 파일이 아니라 폴더 자체)
  5. 목록에 "셀록홈즈 URL 자동검색"이 추가되면 완료! (셀록홈즈에 로그인되어 있어야 작동)
💡 크롬을 켤 때 "개발자 모드 확장 프로그램을 사용 중지" 알림이 뜨면 "확장 프로그램 유지"를 누르세요. · 폴더를 지우면 확장이 멈추니 그대로 두세요. · 자동검색도 셀록홈즈 월 사용량을 1회 차감합니다(직접 검색과 동일).

🔍 기능 1 · 소싱 대시보드 사용법

  1. 셀러라이프 > 상품 소싱 > 카테고리 소싱에서 엑셀 파일 다운로드
  2. 대시보드에 엑셀 업로드 → 자동으로 필터링됨
  3. 왼쪽 사이드바에서 필터 기준(로켓배송비율, 검색량 등) 조정
  4. 결과 표에서:
    • 열 제목의 ▾ → 엑셀처럼 값 선택 필터 · 정렬 (예: 피크월에서 6월만 체크)
    • 카테고리 우측 ▾ → 전체 경로 펼치기
    • 🛒 쿠팡(빨강) → 쿠팡 검색 / 📊 셀록홈즈(파랑) → 키워드 분석
    • 👍/👎 체크 후 📥 CSV 다운로드 → 마킹 포함 저장

🌐 기능 2 · 이미지 번역 사용법

중국 타오바오 상세페이지의 중국어를 한국어로 번역합니다.
  1. OpenAI API 키 준비 (platform.openai.com에서 발급, 각자 본인 키)
  2. 이미지 번역 페이지 → 사이드바에 API 키 입력
  3. 상세페이지 이미지 여러 장 업로드
  4. 🚀 번역 시작 → 진행률 표시
  5. 완료 후 📥 ZIP 다운로드
💡 비용은 각자 본인 OpenAI 키로 청구됩니다 (이미지당 약 $0.08~).

❓ 자주 묻는 문제

Q. run.bat을 눌러도 검은 창이 바로 닫혀요.
→ install.bat을 먼저 실행했는지 확인하세요.

Q. 브라우저가 자동으로 안 열려요.
→ 주소창에 localhost:8501 을 직접 입력하세요.

Q. 셀록홈즈 자동검색이 안 돼요.
→ 확장이 켜져 있는지(chrome://extensions), 셀록홈즈에 로그인됐는지 확인하세요.

Q. 큰 엑셀 파일이 업로드가 안 돼요.
→ 업로드 한도는 1GB입니다. 그보다 크면 알려주세요.
