# 🚀 구매대행 소싱 도구 — 시작 가이드

이 도구는 **① 쿠팡 소싱 키워드 필터링**과 **② 중국어 상세페이지 이미지 번역** 두 가지 기능을 제공합니다.

아래 순서대로 한 번만 설치하면 계속 사용할 수 있습니다.

---

## 📋 준비물 요약

| 순서 | 항목 | 필수 여부 |
|------|------|-----------|
| 1 | Python 설치 | ✅ 필수 (앱 실행용) |
| 2 | 라이브러리 설치 (install.bat) | ✅ 필수 (최초 1회) |
| 3 | 앱 실행 (run.bat) | ✅ 매번 실행 |
| 4 | 셀록홈즈 자동검색 확장 | ⭕ 권장 (셀록홈즈 원클릭 검색) |
| 5 | OpenAI API 키 | ⭕ 이미지 번역 쓸 때만 |

---

## 1단계 · Python 설치 (최초 1회)

1. **https://python.org** 접속 → **Download Python** 클릭
2. 설치 화면에서 **"Add Python to PATH"** 체크 후 **Install**
   - (Microsoft Store에서 "Python 3" 검색 후 설치해도 됩니다)
3. 설치 완료

> 💡 이미 설치되어 있는지 모르겠다면 그냥 2단계로 넘어가세요.
> install.bat이 Python 유무를 자동으로 확인하고, 없으면 안내해줍니다.

---

## 2단계 · 라이브러리 설치 (최초 1회)

1. 받은 ZIP 파일의 **압축을 풉니다**
2. 폴더 안 **`install.bat`** 을 더블클릭
3. 검은 창에서 설치가 진행됩니다 (2~3분)
4. **"설치가 완료되었습니다!"** 메시지가 뜨면 아무 키나 눌러 닫기

---

## 3단계 · 앱 실행 (매번)

1. **`run.bat`** 을 더블클릭
2. 잠시 후 브라우저가 자동으로 열립니다
   - 안 열리면 브라우저 주소창에 **`localhost:8501`** 입력
3. ⚠️ **검은 창(터미널)은 닫지 마세요** — 닫으면 앱이 종료됩니다
4. 사용이 끝나면 검은 창을 닫아 종료

왼쪽 사이드바에 두 페이지가 있습니다:
- **🔍 구매대행 소싱 대시보드**
- **🌐 중국어 → 한국어 이미지 번역**

---

## 🔍 기능 1 · 소싱 대시보드 사용법

1. **셀러라이프 > 상품 소싱 > 카테고리 소싱**에서 엑셀 파일 다운로드
2. 대시보드에 엑셀 **업로드** → 자동으로 필터링됨
3. 왼쪽 사이드바에서 필터 기준(로켓배송비율, 검색량 등) 조정 가능
4. 결과 표에서:
   - **각 열 제목 아래 칸에 입력** → 엑셀처럼 필터 (예: 피크월에 `6월`)
   - **열 제목 클릭** → 정렬
   - **카테고리 ⋯더보기** → 전체 경로 펼치기
   - **🛒 쿠팡** (빨강) → 쿠팡 검색 / **📊 셀록홈즈** (파랑) → 키워드 분석
   - **👍/👎 체크** 후 **📥 CSV 다운로드** → 마킹 포함 저장

---

## ⭐ 4단계 · 셀록홈즈 자동검색 확장 (권장)

설치하면 대시보드의 **📊 셀록홈즈** 버튼 클릭 시 → 셀록홈즈가 열리며 **키워드가 자동 검색**됩니다. (복사·붙여넣기 불필요)

📁 **`셀록홈즈_자동검색_확장`** 폴더 안의 **`설치방법.md`** 를 참고하세요. 요약:

1. 크롬 주소창에 `chrome://extensions` 입력
2. 오른쪽 위 **"개발자 모드"** 켜기
3. **"압축해제된 확장 프로그램을 로드합니다"** → `셀록홈즈_자동검색_확장` **폴더 선택**
4. 완료! (셀록홈즈에 로그인되어 있어야 작동)

> 💡 셀록홈즈는 전용 탭 하나에서 열리므로, 쿠팡 윙은 **처음 한 번만 로그인**하면 계속 유지됩니다.

---

## 🌐 기능 2 · 이미지 번역 사용법

중국 타오바오 상세페이지의 중국어를 한국어로 번역합니다.

1. **OpenAI API 키** 준비 (https://platform.openai.com 에서 발급, 각자 본인 키)
2. 이미지 번역 페이지 → 사이드바에 **API 키 입력**
3. 상세페이지 이미지 **여러 장 업로드**
4. **🚀 번역 시작** → 진행률 표시
5. 완료 후 **📥 ZIP 다운로드**

> 💡 비용은 각자 본인 OpenAI 키로 청구됩니다 (이미지당 약 $0.08~).

## ❓ 자주 묻는 문제

**Q. run.bat을 눌러도 검은 창이 바로 닫혀요.**
→ install.bat을 먼저 실행했는지 확인하세요.

**Q. 브라우저가 자동으로 안 열려요.**
→ 주소창에 `localhost:8501` 을 직접 입력하세요.

**Q. 셀록홈즈 자동검색이 안 돼요.**
→ 확장이 켜져 있는지(`chrome://extensions`), 셀록홈즈에 로그인됐는지 확인하세요.

**Q. 큰 엑셀 파일이 업로드가 안 돼요.**
→ 업로드 한도는 1GB로 설정되어 있습니다. 그보다 크면 알려주세요.
